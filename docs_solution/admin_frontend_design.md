# 优题慧家长系统 - 后台前端方案设计

## 1. 技术栈

- **框架**: Vue3 + TypeScript
- **UI 组件库**: Element Plus
- **构建模板**: Art Design Pro&#x20;
- **状态管理**: Pinia
- **网络请求**: Axios 封装

## 2. 核心功能模块设计

### 2.1 仪表盘 (Dashboard)

- **数据概览**: 平台总注册用户数、今日活跃家长数、VIP/SVIP 开通比例统计。
- **系统监控**: 试卷处理任务队列状态、存储空间使用率。
- **图表展示**: 近30天用户增长趋势折线图。
- 注：请你参考常见后台仪表盘数据，结合此项目（为教育小程序的后台界面，支持家长注册绑定学生，查看学生成绩数据）添加一些常用数据

### 2.2 学生与家长管理模块

- **用户列表**:
  - 支持按手机号、学校、年级、学生姓名进行多条件筛选查询。
  - 列表展示家长账号及其绑定的学生档案信息。
  - 操作栏：查看详情、封禁/解封账号、重置密码。
- **VIP/SVIP 权限管理**:
  - 支持单条或批量开通/调整用户的 VIP/SVIP 状态及有效期。
- **学生档案导入**:
  - 提供标准的 Excel 模板下载链接。
  - 采用 Element Plus 的 `el-upload` 组件实现 Excel 批量导入。
  - 导入结果反馈弹窗：展示成功条数、失败条数及错误原因（如下载失败报告）。

### 2.3 试卷与错题数据管理模块 (核心业务)

- **文件包上传页面**:
  - 分类上传区域：试卷原卷包 (ZIP/PDF/图片)、小题分文件包 (Excel/CSV)、错题本文件包。
  - 上传组件支持大文件分片上传与断点续传。
  - 提交后，展示后端处理的进度条或任务状态列表（解析中 -> 匹配中 -> 完成）。
- **数据校验与修正**:
  - 针对后端返回的“未匹配成功”的试卷/错题记录，提供人工干预界面，手动关联学号或学生姓名。
- **PDF/Word 导出记录**:
  - 列表展示系统为学生生成的试卷、错题集文件链接，支持后台预览和重新生成操作。

### 2.4 课程与自习室管理

- **公益课程管理**:
  - 课程列表：新增、编辑、删除、上下架操作。
  - 课程内容支持富文本编辑及视频链接配置。
- **AI 自习室报名管理**:
  - 查看家长的报名申请列表。
  - 状态管理：待处理、已确认、已拒绝。

### 2.5 客服与互动配置

- **FAQ 管理**:
  - 增删改查常见问题及解答内容。
- **微信群二维码配置**:
  - 后台支持按学校/年级上传并更新微信群二维码图片，供小程序端动态拉取。

## 3. 权限与安全控制

- **RBAC 角色权限控制**:
  - 超级管理员：拥有所有权限。
  - 运营人员：可管理课程、解答配置、处理报名。
  - 数据录入员：仅具有导入学生信息和上传试卷包的权限。
- **操作日志审计**:
  - 展示管理员的操作记录表格，包含操作人、操作时间、IP、操作模块及具体动作。
