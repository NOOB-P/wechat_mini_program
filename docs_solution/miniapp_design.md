# 优题慧家长小程序 - 前端方案设计

## 1. 技术栈
- **框架**: UniApp (Vue3 + TS)
- **UI 组件库**: Wot Design Uni
- **状态管理**: Pinia
- **网络请求**: uniapp 原生 request 封装
- **构建工具**: Vite

## 2. 界面设计与交互逻辑

### 2.1 用户注册与绑定模块
- **界面设计**:
  - 手机号输入框（限11位数字）。
  - 短信验证码输入框及获取按钮（带60s倒计时）。
  - 第三方登录入口（微信/QQ 图标）。
  - 绑定信息表单：级联选择器（省/市/区/学校）、年级选择、班级选择、学生姓名输入框。
- **业务逻辑**:
  - 手机号注册/登录合并，验证码验证成功后，若未绑定则跳转至绑定页。
  - 调用微信 `wx.login` 获取 code，与后端换取 OpenID 进行快捷登录。
  - 绑定页面通过模糊搜索调用后端接口获取学校列表。
  - 提交绑定信息时进行非空校验，成功后在本地缓存（如 `uni.setStorageSync`）中存储 Token 及学生基础档案，并跳转首页。

### 2.2 成绩查询与试卷模块 (免费用户基础功能)
- **界面设计**:
  - 首页展示学生总分及各科概览卡片。
  - 成绩详情页：列表展示历史考试名称、时间、各科成绩及等级。
  - 试卷原卷页：图片或 PDF 预览组件，支持手势缩放。下载按钮位于底部或右上角。
- **业务逻辑**:
  - 首页下拉刷新，调用后端接口获取最新成绩。
  - 试卷查看调用 `uni.previewImage` 或使用 `web-view` 加载 PDF 文件。
  - 下载功能调用 `uni.downloadFile` 和 `uni.saveFile` 将文件保存至本地并提示用户路径。

### 2.3 数据分析模块 (VIP 功能)
- **界面设计**:
  - 采用 Echarts for Weixin 或 uCharts 绘制图表。
  - 成绩构成饼图模块、成绩分布柱状图模块、成绩趋势折线图模块、学习习惯雷达图模块。
- **业务逻辑**:
  - 请求后端数据分析接口，需携带 VIP 鉴权标识。
  - 渲染前将后端返回的统计数据进行格式化，适配图表组件的数据结构要求。
  - 数据均以 ABCDE 等级和百分比呈现，屏蔽具体排名数值。

### 2.4 错题集与打印模块 (SVIP 功能)
- **界面设计**:
  - 错题列表页：支持按科目、考试时间筛选。错题卡片展示题目、原答、正解及解析。
  - 操作按钮：添加笔记、错题归类。
  - 导出/打印页：展示“生成PDF/Word”按钮及“委托打印”入口（选择地址、配送方式）。
- **业务逻辑**:
  - 请求错题数据并支持分页加载。
  - 调用后端生成错题本文件的接口，获取下载链接。
  - 委托打印调用第三方服务 API，拉起微信支付或表单提交页。

### 2.5 AI 课程与自习室模块
- **界面设计**:
  - 课程列表流式布局，展示课程封面、标题、学科标签。
  - 自习室介绍页：图文混排，底部悬浮“立即报名”按钮。
  - AI 学习建议页：分日历视图展示学习计划，并展示推荐资源链接列表。
- **业务逻辑**:
  - 课程播放支持跳转至视频播放页。
  - 报名表单提交至后端记录。
  - 学习建议数据由后端 AI 接口返回，前端负责渲染任务清单，并支持勾选完成状态。

### 2.6 家长咨询与客服
- **界面设计**:
  - 悬浮客服按钮或“我的”页面客服入口。
  - FAQ 列表页：手风琴折叠面板。
  - 微信群引导：展示二维码。
- **业务逻辑**:
  - 调用小程序原生客服 `button open-type="contact"`。
  - 长按保存或识别微信群二维码功能。

## 3. 安全与权限
- 所有请求统一在封装的拦截器中加入 Token。
- 对涉及隐私的请求数据采用加盐签名防篡改。
- 页面守卫：在关键页面的 `onShow` 生命周期判断用户 VIP/SVIP 状态，无权限则弹出开通引导蒙层。
