# 学校导入说明
为了能够在导入后学生自动绑定学校班级，请遵循以下格式（系统会自动生成唯一序列号）。

## 通用要求
所有学校相关模板都必须完整填写以下基础字段：

- `省份`
- `城市`
- `区县`
- `学校`

其中 `区县` 为选填项；如果学校没有完善区县信息，通常按高中学校处理，可留空。

### 批量导入学校(最小单位 学校)
如果你只是希望批量导入学校，使用 `学校模板.xlsx`。

模板列顺序：

1. `省份`
2. `城市`
3. `区县`
4. `学校`

### 批量导入学校班级(最小单位 班级)
如果希望批量导入学校，并且班级一起导入，使用 `学校-班级模板.xlsx`。

模板列顺序：

1. `省份`
2. `城市`
3. `区县`
4. `学校`
5. `年级`
6. `班级`

### 批量导入学生(最小点位 学生)
如果希望连同学生一起导入，使用 `学校-班级-学生模板.xlsx`。

模板列顺序：

1. `省份`
2. `城市`
3. `区县`
4. `学校`
5. `年级`
6. `班级`
7. `学号`
8. `姓名`
